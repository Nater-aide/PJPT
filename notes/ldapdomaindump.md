# LdapDomaindump
This tool is built into Kali  
quick and easy to run

1. Make a directory
2. ```sudo ldapdomaindump ldaps://(domain controller IP) -u 'MARVEL\fcastle' -p Password1```
3. This will dump the information in the domain. Search through .html files for clean look

Interesting files
- domain_policy.html
- domain_computers.html -- incredibly useful
- domain_users_by_group.html
