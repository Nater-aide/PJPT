# Physical AD Components
- Domain controllers
  -  most important
  -  What controls everything
  -  Provides authorization and authentication
- AD DS Data store
  -  stores database files
  -  It contains the NTDS.dit file
 
# Logical AD Components
- Schema
  - Rulebook or a blueprint
  - Defines every type of object that is stored
  - Class Object
    - User
    - Computer
  - Attribute object
    - Display name
- Domain
  - Used to group and manage objects
  - Think of it as a boundry
- Tree
 - Hierarchy of domains
- Forests
  - Colletion of trees
